# Zen Browser for Windows 8+

Zen Browser is a modern web browser based on the Firefox engine, designed with a focus on simplicity, performance, and distraction-free browsing. It offers a clean, minimal interface, fast performance, and a set of features aimed at improving the overall web experience.

Zen Browser is not officially supported on older operating systems such as Windows 8/8.1, as it relies on newer web technologies. This repository provides a modified, up-to-date version of Zen Browser, adapted to run on Windows 8+, allowing users to benefit from the latest features and security updates despite system limitations.
